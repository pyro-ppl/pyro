Bayesian Inference for Pytorch

## Installation

First install [Pytorch](http://pytorch.org/).

Install via pip:
```sh
pip install pyroppl
```

Install from source:
```sh
git clone git@github.com:uber/pyro.git
cd pyro
pip install .
```
